Amazon Web Data Scraper

This project involves creating a web scraper using Python to extract product data from Amazon. The scraper collects various details, such as product names, prices, ratings, and descriptions, by navigating through the website's structure. The data is then stored for analysis, allowing users to monitor trends, compare products, or track changes over time. This tool enhances the ability to gather insights from Amazon's extensive product listings efficiently.

This project involves developing a Python application to scrape sneaker data from Amazon, specifically targeting men's Nike sneakers. The main objective is to collect real-time information on various sneakers, including details such as brand, title, price, color, review count, and availability.

Key Features:
1. Data Extraction: Utilizes the BeautifulSoup and requests libraries to extract relevant sneaker information from product pages. The data collection includes:
- Brand Name: Extracts the brand associated with each sneaker.
- Product Title: Captures the title of the sneaker for identification.
- Price: Retrieves the current price of the sneaker.
- Color: Identifies the color options available, if mentioned.
- Review Count: Collects the total number of reviews to gauge product popularity.
- Availability: Checks if the sneaker is currently available for purchase.
2. Dynamic Pagination: Implements a loop to navigate through multiple pages of search results on Amazon, allowing for comprehensive data collection.
3. Data Organization: Organizes the scraped data into a structured format using a Pandas DataFrame, facilitating further analysis and visualization.
4. CSV Export: Once the scraping process is complete, the collected data is saved to a CSV file, enabling easy access and analysis in tools like Excel or Tableau.

Technologies Used:
1. Python: The primary programming language used for developing the scraper.
2. BeautifulSoup: A library for parsing HTML and XML documents, crucial for extracting data from web pages.
3. requests: A library for making HTTP requests to retrieve web page content.
4. Pandas: A data manipulation and analysis library used to store and export the scraped data.
5. NumPy: Used for handling numerical data, particularly in managing missing values.

This project demonstrates proficiency in web scraping techniques, data manipulation, and the ability to extract actionable insights from e-commerce platforms while adhering to best practices in data handling and scraping guidelines.
