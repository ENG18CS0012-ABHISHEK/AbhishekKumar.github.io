Airbnb Data Analysis and Visualization

This project involves analyzing a dataset from Airbnb to uncover insights about listings, prices, and customer preferences. The data undergoes a cleaning process to ensure accuracy and consistency, after which it is visualized using Tableau to create interactive dashboards. These visualizations help highlight trends in rental prices, geographic distribution of listings, and other key metrics, enabling better understanding of the Airbnb market and aiding stakeholders in making informed decisions.

//View Tableau Airbnb Visualization dashboard//

This project involved analyzing an Airbnb dataset sourced from Kaggle to uncover and visualize essential insights. Due to the file size exceeding 25MB, the dataset was divided into two parts: Airbnb Project.part1 and Airbnb Project.part2 for GitHub uploads. I conducted thorough data cleaning to address missing values and inconsistencies. After this process, I utilized Tableau to create interactive dashboards that showcase various features of Airbnb listings.

The dataset contains information such as listing URLs, host details, location coordinates, pricing, amenities, availability, and guest reviews. Key insights derived from the visualizations include pricing trends, popularity based on location, host responsiveness, and review ratings for different properties.

Technologies Used:
1. Data Cleaning: Microsoft Excel
2. Visualization: Tableau
3. Dataset: Airbnb listings dataset from Kaggle
